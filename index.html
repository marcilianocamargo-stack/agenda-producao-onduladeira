1. Programação do dia

⏳ Buscando a programação do dia...
Sincroniza com o programacao-hoje.xlsx publicado nesta mesma pasta do GitHub Pages (mesmo arquivo do Romaneio e das Etiquetas). É seguro sincronizar quantas vezes quiser: OF nova entra na fila; OF que você já moveu para "Aguardando" ou "Produzidos" continua lá; OF que sumir da planilha (expedida/cancelada) sai das listas daqui também. Nada que você já organizou é perdido.

2. Lançar pedido manual

3. Produção por cliente

Arraste um cliente (⠿ no cabeçalho) para mudar a ordem de fabricação entre clientes. Toque no nome do cliente para abrir e ver/reordenar os pedidos daquele cliente. "Aguardar" tira da projeção sem apagar (ex: esperando fechar carga); "Produzido" registra a fabricação (pode ser parcial).
Pedidos que existem mas não devem entrar na projeção de capacidade ainda (ex: aguardando fechar carga para uma região). Não contam nos dias da agenda. Toque em "Liberar p/ fila" quando puder produzir.
Já fabricados, mas continuam aqui até serem expedidos (a OF permanece controlada, igual você já faz na planilha). Remova só quando o Romaneio confirmar a expedição.

4. Publicar

Como criar o token (1x): no GitHub, vá em Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Marque "Only select repositories" → escolha agenda-producao-onduladeira → em Permissions, ative Contents: Read and write. Copie o token e cole acima.

O token fica salvo só neste navegador/dispositivo (nunca é enviado a mais ninguém além do próprio GitHub). Remover token salvo.
Alternativa sem token: baixa o arquivo pra você subir manualmente no GitHub, igual ao fluxo do Romaneio.
